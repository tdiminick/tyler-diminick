How I Work

I'm a full-stack engineer who's comfortable owning a problem end to end — data modeling, API design, frontend architecture, infrastructure, deployment. My go-to stack is Vue 3 on the frontend, Node.js or Python on the backend, and PostgreSQL for data. I've built and deployed on AWS and GCP at scale, and I'm just as happy wiring up a VPS for a side project. I work primarily in JavaScript/TypeScript but reach for whatever tool fits the job.

Professionally, I've spent a decade shipping software — from monitoring hundreds of thousands of EC2 instances at Amazon to architecting core product features at early-stage startups. Outside of work, I build products that scratch my own itch. The Career Actor and Improv Jam Manager both came from real problems I ran into as a performer. I'd rather build something useful than something impressive.
